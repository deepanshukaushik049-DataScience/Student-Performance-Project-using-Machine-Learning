## Overview

This project aims to predict student exam performance based on various academic and lifestyle factors such as study hours, attendance, sleep, and more.

It demonstrates the complete Data Science workflow:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning Model Building
